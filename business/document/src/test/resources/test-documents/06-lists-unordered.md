# 无序列表与复选框测试

## 短横线无序列表

以下是一个标准的无序列表，使用短横线作为标记：

- 支持多种文档格式：PDF、DOCX、Markdown、TXT
- 自动提取文档结构和章节信息
- 基于 Elasticsearch 的全文检索能力
- 支持 Neo4j 知识图谱关联查询
- 提供 RESTful API 和 WebSocket 实时推送

## 星号与加号列表

* 星号列表项 —— 用于强调重要特性
* 支持自定义提取规则和插件扩展
* 提供可视化的文档结构树展示

+ 加号列表项 —— 表示新增或待办事项
+ 接入 LLM 进行二次语义判定
+ 实现结构节点的智能归并

## 圆点符号列表

• 中文圆点符号也常见于中文文档
• 支持混合排版场景
• 容错处理非标准格式
