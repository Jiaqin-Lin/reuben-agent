# 朴素标题启发式判定测试

以下是一些看起来像标题但没有正则标记的短行。它们通过空白行隔离、名词短语特征、后跟内容行等线索来触发朴素标题判定。

## 触发朴素标题判定的示例

概述

本模块负责文档的完整生命周期管理，包括上传、解析、结构提取和索引存储。

核心概念

在深入实现细节之前，我们需要理解以下核心概念：文档结构信号、结构节点、归并管线。

技术架构

系统采用分层架构，自上而下分为接入层、业务层、基础设施层。每一层有明确的职责边界。

关键指标

系统需要满足以下性能指标：文档解析延迟 < 5秒，结构提取准确率 > 95%，索引查询延迟 < 200ms。

## 边界情况

短行

短行后面如果没有内容行，或者没有空白行隔离，可能不会触发判定。

又一行

这个短行后面是另一个短行而非正文内容，也可能影响判定结果。
